Ohio is flexible and high performance creative WooCommerce theme.
Author: colabrio
Author URI: http://clbthemes.com/
Support Website: https://colabrio.ticksy.com/
Documentation Website: https://docs.clbthemes.com/ohio/


== Installation ==
Manual installation:

1. Upload the `ohio` folder to the `/wp-content/themes/` directory

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress;
2. See Theme Settings to change theme specific options;

== Custom Logo ==
This theme supports custom logo for the header part. 
You can change or remove it from Theme Settings -> Branding.
